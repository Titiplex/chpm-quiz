# Future couche HTTP

Endpoints ciblés pour la suite :

- `GET /api/session/demo-role` pour remplacer le faux utilisateur local.
- `GET /api/questionnaires/:id` et `POST /api/questionnaires` pour l’admin.
- `POST /api/invitations` pour le modérateur.
- `GET /api/respondent/session/:code` et `PATCH /api/respondent/session/:code/draft` pour le répondant.
- `GET /api/statistics` pour l’administrateur.
